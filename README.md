# AutoBank

Proyecto local para la carga, lectura y analisis de extractos bancarios.

## Estructura

- `AGENTS.md`: organizacion de agentes y subagentes para el desarrollo del proyecto
- `PROJECT_GUIDELINES.md`: pautas tecnicas y convenciones del proyecto
- `PRODUCT_REQUIREMENTS.md`: definicion funcional inicial
- `web/`: aplicacion frontend en React + TypeScript + Vite

## Frontend

Para arrancar la aplicacion:

```bash
cd web
npm install
npm run dev
```
