# AutoBank Web

Frontend de AutoBank construido con React, TypeScript y Vite.

## Scripts

```bash
npm install
npm run dev
npm run start:local
npm run build
npm run lint
```

`npm run start:local` es el comando recomendado para desarrollo local en AutoBank.
Si lo ejecutas en primer plano dentro de `web/`, `Ctrl + C` detiene Vite y libera el puerto `3000`.

## Objetivo Inicial

Esta aplicacion servira como base para:

- subir extractos bancarios
- leer y normalizar movimientos
- visualizar resumenes del periodo
- evolucionar hacia analitica mensual y anual
